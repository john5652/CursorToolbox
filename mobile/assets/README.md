# Assets Directory

This directory contains app assets including icons, splash screens, and images.

## Required Assets

### App Icon
- **File**: `icon.png`
- **Size**: 1024 x 1024 pixels
- **Format**: PNG
- **Requirements**: 
  - No transparency
  - Square image
  - High quality

### Adaptive Icon (Android)
- **File**: `adaptive-icon.png`
- **Size**: 1024 x 1024 pixels
- **Format**: PNG
- **Requirements**:
  - Foreground image only
  - Safe area: 66dp from edges
  - No transparency

### Splash Screen
- **File**: `splash.png`
- **Size**: 1242 x 2436 pixels (recommended)
- **Format**: PNG
- **Requirements**:
  - Will be centered and scaled
  - Background color set in app.json

### Favicon (Web)
- **File**: `favicon.png`
- **Size**: 48 x 48 pixels (minimum)
- **Format**: PNG or ICO

## Creating Assets

### Using Design Tools
1. Create designs in Figma, Sketch, or Adobe XD
2. Export at required sizes
3. Optimize images (use tools like ImageOptim or TinyPNG)

### Using Online Tools
- [App Icon Generator](https://www.appicon.co/)
- [Icon Kitchen](https://icon.kitchen/)
- [MakeAppIcon](https://makeappicon.com/)

## Notes

- All assets should be optimized for file size
- Use PNG format for best quality
- Test icons on actual devices
- Ensure icons look good at small sizes
